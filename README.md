### 二、文件提交规范

￼￼
每个小组目录下已预设4个子文件夹，請将对应类型的作品放入指定文件夹，文件名统一规范，便于识别：

##### 1.初赛PPT

 存放小组初赛展示PPT，文件名格式：【小组名】-初赛PPT-作品名.pptx（例：【瞬忆•智能记忆】-初赛PPT-智能记忆系统.pptx）

##### 2.决赛PPT

存放小组决赛展示PPT，文件名格式：【小组名】-决赛PPT-作品名.pptx

##### 3.视频资料：

存放作品演示视频、答辩视频等，文件名格式：【小组名】-视频-视频类型.mp4（例：【缝合怪】-视频-作品演示.mp4）；若视频超过100MB，请勿直接上传，可上传至网盘，在该文件夹内创建「视频链接说明.txt」，填写网盘链接、提取码及视频说明。

##### ￼4.其他补充资料：

存放作品其他需要的资料，文件名格式：【小组名】-补充资料-资料名称.后缀（例：【智能花盆】-补充资料-设计说明书.docx）

### 三、提交方式

##### 方式1：网页端直接上传（推荐，最简便）

1. 打开本GitHub仓库链接，登录自己的GitHub账号；
2. 找到自己小组的文件夹，点击进入（例：点击「瞬忆•智能记忆」）；
3. 点击对应子文件夹（如「初赛PPT」），再点击右上角「Add file」→「Upload files」；
4. 拖拽文件至上传框，或点击「choose your files」选择本地文件；
5. 下拉至页面底部，在「Commit changes」下方填写提交说明（例：瞬忆•智能记忆 提交初赛PPT）；
6. 选择「Commit directly to the main branch」，点击「Commit changes」，完成提交。

##### 方式2：Fork + Pull Request（规范，适合熟悉Git的小组）

1. 点击仓库右上角「Fork」，将仓库复制到自己的GitHub账号下；
2. 在自己的Fork仓库中，按上述规范上传文件并提交；
3. 点击「Compare & pull request」，填写提交说明，提交合并请求；
4. 等待管理员审核通过后，即可完成提交。
