# 智能矫姿颈枕

配合智能矫姿颈枕设备使用的微信小程序，实时监测颈部压力数据，调节气囊支撑、按摩模式与温度设置，追踪健康趋势，帮助改善颈椎姿态和睡眠质量。

---

## 功能概览

### 首页 — 实时压力监测

打开小程序即可看到颈枕的压力分布图。

枕头图形上覆盖了 5 个传感区域：**左侧护翼、右侧护翼、后颈承托、左下承托、右下承托**。每个区域会实时显示当前压力数值，颜色从绿色渐变到黄色、橙红色，分别代表**舒适、适中、偏紧**三种状态。

页面下方会同步展示：

- **最高压力**与**受力区域数量**
- **综合姿势评分**（综合左右平衡与舒适度计算，满分 100）
- **小贴士**——根据当前压力分布自动给出的改善建议

> 提示：点击「开始监测」后，数据每 0.3 秒自动刷新。如需查看某一时刻的压力数据，点击「暂停监测」定格画面。

---

### 控制页 — 调节颈枕

进入控制页面，可以精细调节颈枕的支撑方式和按摩功能。

#### 快捷模式

一触切换到最适合当前场景的支撑配置：

| 模式 | 适用场景 | 说明 |
|------|---------|------|
| 办公 💻 | 长时间工作 | 轻度支撑，保持颈椎自然曲度 |
| 睡眠 🌙 | 夜间休息 | 柔和包裹，减少翻身时打扰 |
| 旅行 ✈️ | 出行途中 | 稳固贴合，抵抗颠簸 |
| 放松 ☕ | 休息间隙 | 轻柔支撑 + 自动开启按摩 |

#### 气囊调节

5 个独立气囊支持 0-100% 充气值的精细调节：

- **左侧护翼 / 右侧护翼**：控制头部左右两侧的包裹力度
- **后颈承托**：颈后支撑高度
- **左下承托 / 右下承托**：下巴两侧的支撑强度

点击「**全部复位**」可将所有气囊归零；点击「**智能调节**」可根据近期压力数据自动分配充气值。

#### 按摩功能

开启按摩后，可选择：

- **模式**：揉捏 / 敲击 / 振动 / 脉冲
- **强度**：1-5 级，逐级增强
- **定时**：15 / 30 / 45 分钟，到时自动停止

> 注：放松模式会自动开启揉捏按摩，切换到其他模式后按摩保持当前设置。

#### 温度控制

打开加热后，可设置目标温度（35°C - 45°C），或直接选择快捷温度：**36°C / 38°C / 40°C / 42°C**。

### 健康页 — 查看健康数据

在「健康」页面可以查看压力历史趋势、睡眠报告和姿势评估。

#### 压力趋势

支持切换「日 / 周 / 月」三个时间维度：

- **折线图**：展示压力变化曲线，橙色虚线为 60 压力预警线，超出该线的区域以橙红色标注
- **柱状图**：每个时间节点显示具体数值和压力等级
- **摘要卡片**：快速查看峰值、低值、预警次数和趋势变化

#### 睡眠报告

展示昨夜睡眠数据：

- 总时长、深睡时长、翻身次数、晨起颈部感觉
- 睡眠阶段分布：深睡 / 浅睡 / REM / 清醒

#### 姿势评估

综合评分以环形仪表盘展示，同时列出：

- **左右平衡**：左右两侧压力差异百分比
- **前后支撑**：颈椎前后支撑均匀度
- **压力均匀度**：5 个区域压力分布的均匀程度

## 快速开始

### 1. 下载与安装

在微信中搜索并打开本小程序。

### 2. 连接颈枕设备

点击首页顶部的「设备连接」卡片，小程序会自动搜索附近的颈枕设备。搜索到后点击设备名称完成配对连接。

> 首次连接需要在手机蓝牙设置中允许微信访问蓝牙。

### 3. 开始使用

连接成功后，首页会自动显示实时压力数据。根据当前需要，进入「控制」页面调节支撑模式，或在「健康」页面查看数据趋势。

## 常见问题

**Q：首页数据不更新怎么办？**

A：确认颈枕设备已开机且电量充足。点击首页「暂停监测」再点击「开始监测」重置数据流。如问题持续，尝试在小程序设置中重新配对设备。

**Q：按摩功能无法开启？**

A：部分型号颈枕在电量低于 10% 时会限制按摩功能，请先充电。

**Q：加热温度感觉不明显？**

A：目标温度为颈枕内部温度传感器读数，实际体感会因环境和佩戴方式有所差异。建议从 38°C 开始尝试，逐步调整到舒适区间。

**Q：姿势评分是如何计算的？**

A：综合评分基于「左右压力平衡（占 60%）」和「平均压力舒适度（占 40%）」两个维度计算，得分越高表示姿势越健康。

**Q：数据会自动保存吗？**

A：每日的压力摘要、佩戴时长、姿势评分等数据会加密存储在本地。切换设备或重装小程序前，请前往「设置」中备份数据。
